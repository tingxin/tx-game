🎯 AI 图片分析器 - 启动说明

📋 项目结构：
├── index.html      # 前端页面
├── style.css       # 样式文件  
├── script.js       # 前端逻辑
├── app.py          # Flask 后端
├── run.py          # 启动脚本
├── requirements.txt # Python 依赖
└── 启动说明.txt    # 本文件

🚀 快速启动：

1. 安装 Python 依赖：
   pip install -r requirements.txt

2. 配置 AWS 凭证（必需）：
   
   方式A - 环境变量（推荐）：
   export AWS_ACCESS_KEY_ID=your_access_key_id
   export AWS_SECRET_ACCESS_KEY=your_secret_access_key  
   export AWS_REGION=us-east-1
   
   方式B - 直接在 config.py 中设置：
   编辑 config.py 文件，取消注释并填入你的凭证

3. 测试 AWS 连接（可选）：
   python test_aws.py

4. 启动服务：
   python run.py
   
   或者直接：
   python app.py

5. 打开浏览器访问：
   http://localhost:5000

✨ 功能特性：
- 拖拽上传图片
- 实时预览
- AWS Bedrock Nova Pro AI 分析
- 结果复制功能
- 响应式设计
- 专业 UI 界面

⚙️ 技术栈：
- 前端：纯 HTML + CSS + JavaScript
- 后端：Python Flask
- AI：AWS Bedrock Nova Pro
- 样式：原生 CSS + Font Awesome 图标

🔧 注意事项：
- 确保 AWS 账户有 Bedrock 访问权限
- 图片大小限制 10MB
- 支持 JPG、PNG、GIF 格式
